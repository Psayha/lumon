# Админ-панель Lumon Platform

## Структура

```
adminpage/
├── AdminPage.tsx          # Главный компонент
├── components/
│   └── AdminLogin.tsx    # Компонент авторизации
├── tabs/
│   ├── CompaniesTab.tsx   # Таб с компаниями
│   ├── LegalDocsTab.tsx  # Таб с юр документами
│   └── AIDocumentsTab.tsx # Таб с документами для ИИ
├── main.tsx              # Entry point
├── index.html            # HTML шаблон
└── vite.config.ts        # Конфигурация Vite
```

## Функционал

### 1. Авторизация
- Простая форма входа (admin / admin для разработки)
- Сохранение токена в localStorage

### 2. Таб "Компании"
- Список всех компаний
- Поиск по названию
- Раскрывающийся список пользователей компании
- Отображение ролей пользователей

### 3. Таб "Юр документы"
- Список юридических документов
- Редактирование документов
- Обновление текста для модалки AgreementModal

### 4. Таб "Документы для ИИ"
- Загрузка документов (PDF, DOC, DOCX, TXT)
- Список загруженных документов
- Удаление документов
- Статусы обработки
